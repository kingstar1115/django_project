django admin simpleui
-----
django admin 美化，采用layui 作为基本框架，改写大部分页面

+ 安装
> pip install django-simpleui
+ 项目 settings.py 的INSTALLED_APPS 第一行加入
> `simpleui`

+ 美化页面
 + 登录页
 + 主页
 + 列表页
 + 编辑页
 + 历史
